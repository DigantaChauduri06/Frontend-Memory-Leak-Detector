# Subscription Scanner UI

Fullstack RxJS Subscription Scanner — find and fix unmanaged subscriptions in Angular/RxJS codebases.

## Quick Start

### Backend (port 4000)

```bash
cd backend
npm run dev
```

### Frontend (port 5173)

```bash
cd frontend
npm run dev
```

## Usage

1. Start both servers
2. Open http://localhost:5173
3. Enter the absolute path to an Angular/RxJS project
4. Click **Scan**
5. Browse, filter, and expand issues to see fix suggestions
